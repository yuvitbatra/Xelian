# calc-mcp

MCP server for math: safe expression evaluation and list statistics.

Pure Python standard library — no dependencies, no API keys.

## Usage

```bash
xelian run xelian/calc-mcp      # stdio MCP server
xelian gateway add xelian/calc-mcp      # or serve it via the gateway
```

Point any MCP client at it, or use the Xelian gateway for a single endpoint.
