# time-mcp

MCP server for time: current time, epoch and ISO-8601 conversion.

Pure Python standard library — no dependencies, no API keys.

## Usage

```bash
xelian run xelian/time-mcp      # stdio MCP server
xelian gateway add xelian/time-mcp      # or serve it via the gateway
```

Point any MCP client at it, or use the Xelian gateway for a single endpoint.
