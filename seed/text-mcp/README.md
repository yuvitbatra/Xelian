# text-mcp

MCP server for text: regex search/replace and unified diffs.

Pure Python standard library — no dependencies, no API keys.

## Usage

```bash
xelian run xelian/text-mcp      # stdio MCP server
xelian gateway add xelian/text-mcp      # or serve it via the gateway
```

Point any MCP client at it, or use the Xelian gateway for a single endpoint.
